Alist（云盘）

一个支持多种存储的文件列表程序
支持多个存储提供商，包括本地存储、阿里云盘、OneDrive、Google Drive 等
支持 WebDAV

进入宝塔面板 -> 网站 -> 添加站点

  • 域名：填写列表程序的将要使用域名
  • FTP：不创建
  • 数据库：不创建
  • PHP版本：纯静态

    创建好后直接进入网站根目录，删除里面的404.html、index.html

    安装Alist

去AList Release下载对应系统的程序后端，上传至根目录并解压

获取密码

添加反向代理以及其他设置

  • 开放5244端口

  • 在网站配置文件的 server 字段中添加

  • 网站设置 -> 添加反向代理 -> 代理名称alist -> 目标URLhttp://127.0.0.1:5244 -> 发送域名$host

如需申请SSL证书，可先在SSL选项中申请证书，然后在设置反向代理。或者设置反向代理之后，关闭反向代理功能，申请SSL证书后再次开启代理。

Navidrome（音乐服务器）

一个开源的基于 Web 的音乐收藏服务器和流媒体。它使您可以自由地从任何浏览器或移动设备收听您的音乐收藏。这就像您的个人 Spotify！

外部集成

配置 Navidrome 可以从 Last.fm 和 Spotify 获取专辑信息和图像

Last.fm
提交表单后，可以从 API account Created 页面获取 API Key 和 Shared Secret

Spotify
单击 Spotify 的开发者仪表板中的CREATE AN APP -> CREATE 获取 ND_SPOTIFY_SECRET

Lsky Pro（图床）

兰空图床可以帮您保管大量无处安放的图片，数据可以自由选择储存驱动，支持主流第三方储存。作为一个助手，它不仅可以将您把图片以指定规则存放在指定位置，还有更多强大的功能来帮助您处理这些图片。

准备

  • PHP 版本 7.3
  • MySQL 版本 ≥ 5.5
  • nginx ≥ 1.18
  • PDO 拓展
  • ZipArchive 支持
  • fileinfo 拓展
  • curl 拓展
    注：推荐使用 PHP 7.3, 如果使用 FTP 功能，需要开启 PHP 的 FTP 拓展。fileinfo需要在PHP设置中安装，其他宝塔已经默认安装了。

在宝塔面板新建站点

  • 填写域名
  • 填写MySQL信息
  • PHP版本选择 7.3
  • FTP开启
  • 选择项目根目录
  • 上传源码包到站点目录下，并解压文件到项目根目录
  • 修改运行目录项目根目录下的 /public，运行目录为/

    设置伪静态

  • 在宝塔中开放端口
  • 访问绑定的域名或ip+prot

    安装fileinfo扩展

    PHP设置->安装扩展->选择fileinfo扩展->检查readlink、symlink、putenv、getenv、exec、bash_exec、chmod、chown、fileperms函数没有被PHP禁用函数

    配置云服务OSS

    阿里云OSS
    腾讯云OSS
    FTP 功能

OSS配置自定义域名

文件上传到OSS后，OSS会使用Bucket默认域名生成文件访问地址。使用浏览器访问该地址时，会触发强制下载行为。如果您的业务不希望暴露Bucket默认域名或有在浏览器中预览文件的需求，您需要为Bucket绑定自定义域名，然后通过自定义域名访问该Bucket中的文件。绑定自定义域名后，不影响继续使用OSS提供的域名。

  • 进入阿里云OSS对象储存控制台，选择Bucket配置，域名管理
    阿里云绑定自定义域名至Bucket默认域名

设置runtime目录 0755 权限
