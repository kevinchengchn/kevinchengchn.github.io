前言

免费的域名，你还在等什么？
EU.ORG 永久免费域名，记录一下申请和绑定Cloudflare的流程。
Freenom 免费的域名，一年一次续期。

EU.ORG

免费域名的先驱之一，来感受一下九十年代后期的网站风格吧！该网站自 1996 年以来一直拥有存在，他们的免费子域名由一群欧洲黑客运营，主要面向非营利组织和个人，但也接受小型商业网站。
优点：永久免费，不需要像freenom每年续期一次
缺点：不是顶级域名（这又有什么关系呢）；申请周期有点长，需要耐心等待

注册流程

地址生成器

Register -> 填写地址生成器的信息（最好是英国地址）-> 邮箱激活 -> 用户名：XXX-FREE -> New Domain

填写申请的域名Complete domain name（格式：xxxxxx.eu.org） ->Name servers 选择 server names

Name1: athena.ns.cloudflare.com Name2: jay.ns.cloudflare.com (这个先暂填，后续域名绑定Cloudflare后再修改)

Submit

后续就是等待邮件通知了

绑定Cloudflare

Cloudflare

域名申请成功 -> Cloudflare添加站点 -> 选择免费 -> 添加 Cloudflare 名称服务器 -> eu.org中-选择自己的域名-Nameservers -> 将Cloudflare两条记录更换之前填写的Name1 和 Name2 -> 等待生效

Freenom

freenom是世界上第一个也是唯一的(no no no, eu.org 才是真神)免费域名提供商。 我们的使命是帮助各国人在网上发展自己的数字经济。 免费域名的工作机制与任何其他域名完全一致。 你可以用它作为您的网站，博客，电子邮件帐户或其他！ 您可以选择运行免费域名与网址转发，免费Freenom DNS服务或您自己的DNS（域名服务器）。
缺点：能申请，但很麻烦
