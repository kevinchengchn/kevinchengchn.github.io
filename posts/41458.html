哪一桌没有你的菜

发表于2021-03-19|更新于2021-05-21|🖖🏼
|字数总计:2.1k|阅读时长:6分钟|阅读量:

个人菜谱

还在学习中…

素

烧茄子

茄子、蒜瓣
辅料： 酱油、糖、盐、水、油

  1. 先将茄子去皮，切成块。放入一大碗内，均匀的撒上一层盐，边撒边搅拌。然后放到一边备用。 　

  2. 这时候，你可以剥蒜。将蒜瓣切成蒜绒放在碗里，倒上一点酱油。加一点白糖和水，凑够一碗。香菜切末（如果你是熟练工的话。因为茄子还要淹一会）大概30分钟后，使力气的活该开始了。 　

  3. 抓起大碗里面的茄子，拼命的攥~~尽量把水攥干。攥的越干，烧出来的茄子越香。待油8分热的时候，将攥成球球的茄子下锅。炒到茄子变色倒入刚才按照秘方配置的汤料。烧开后关小火，记得还要翻翻。等汤收的差不多了，关火倒入香菜末，翻锅数下，装盘

凉拌秋葵、花菜

秋葵、蒜蓉
辅料： 生抽、盐、油 （小米椒、青椒）

  1. 秋葵洗干净去蒂，烧一锅开水滴入几滴植物油，放小半勺盐，秋葵入锅烫一两分钟捞出装盘

  2. 备好蒜蓉，适量生抽，盐，加少量清水搅拌成料汁。热锅放油，爆香蒜蓉，倒入料汁炒香，淋在装盘的秋葵上

汤

鲫鱼豆腐汤

鲫鱼1条、 豆腐1块、姜3片、葱3段、盐、胡椒粉、料酒

  1. 鲫鱼开膛去内脏，去鳞去鳃（鱼老板完成），洗净，抹干，用盐和料酒稍腌待用，豆腐切成1厘米厚的块。锅烧热，放入少量油，将鲫鱼放入，煎至两面呈金黄色。
  2. 加入葱姜，放入足够开水（4碗左右）
  3. 水烧开后，放入豆腐用大火煲10分钟。
  4. 汤色会变雪白，再转中火煲40分钟。加入盐、胡椒粉
    tips：鱼煎好后要加热水，汤里可以放几粒花椒去腥，这样做出来的鱼汤才不会腥。

荤

卤牛肉套餐

米饭230g（生米80g），熟牛肉100g，绿叶菜80g，坚果10g，小水果若干

热量：

番茄牛肉

牛肉、胡萝卜、土豆、番茄、鸡蛋、葱、水淀粉

  1. 胡萝卜、土豆、牛肉、葱切丁，番茄切块；
  2. 牛肉加入鸡蛋、油、黑胡椒腌制；
  3. 葱白爆香后加入牛肉；
  4. 煎2分钟后加入胡萝卜和番茄、水；
  5. 最后加入生抽、盐、水淀粉收汁；
  6. 加葱花出锅；

热量：

番茄鱼柳盖饭

巴沙鱼（鲷鱼、龙利鱼、罗非鱼）、番茄、盐、黑胡椒、糙米饭

  1. 巴沙鱼柳加盐和黑胡椒腌制；
  2. 起锅烧油、葱姜蒜爆香；
  3. 放入巴沙鱼翻炒2分钟；
  4. 加入西红书、盐、生抽翻炒后小火收汁；

热量：230kcal；碳水：20g；蛋白质：26g；脂肪：5g；

虾仁鸡蛋羹

3个鸡蛋、100g虾仁、1.5倍鸡蛋液的水、西葫芦

  1. 虾仁洗净后放盐腌制、西葫芦切片
  2. 蒸鸡蛋液4分钟后加入西葫芦和虾仁
  3. 再蒸6分钟后加一勺生抽出锅

*可搭配一根玉米提供碳水
热量：340kcal；碳水：19g；蛋白质：26g；脂肪：18g；

香煎鸡胸肉

鸡胸肉、西兰花、小葱

  1. 鸡胸肉半解冻切片，加入小苏打腌制2分钟后洗净
  2. 加油、淀粉、黑胡椒、生抽、辣椒酱、蒜蓉、剁椒酱腌制
  3. 起锅烧油，葱白爆香后平煎鸡胸肉
  4. 煎至金黄后加入西兰花
  5. 最后撒上葱段

热量：

啤酒花甲

花甲、蒜、葱、杭椒、小米椒、姜、生抽、醋、料酒、啤酒

  1. 处理花甲：清水放一点盐清洗外壳、加香油泡20分钟
  2. 花甲放入锅中放水，打开大火，煮开
  3. 把煮开的花甲捞出来，此时的花甲应该都开口了，把这些花甲都捞出来用凉水在清洗2遍，因为壳子里面有可能还会有部分淤泥
  4. 开火把锅烧热，放油，放入姜丝，蒜泥炒香，放花甲，翻炒一分钟，放料酒，然后放生抽，根据自己口味放；翻炒一分钟，放辣椒、放啤酒
  5. 快出锅的时候放盐，注意，盐要少放，一点一点放，尝一下，觉得淡了再继续放一点，然后放一点醋提鲜，最后放葱，出锅
红烧大虾

大虾、生姜、蒜瓣、料酒、生抽、老抽、青椒、香葱

  1. 基围虾冲洗干净, 姜切片, 葱切段
  2. 起油锅中量油7成热，下葱姜爆香
  3. 虾壳泛红加入料酒、生抽、老抽、青椒、香葱
  4. 中火收汁
白灼大虾

大虾、盐、姜、葱、料酒、蘸料（小米椒、蒜蓉、葱花、生抽\老抽，可爆炒提香）

  1. 水烧开、加入葱、姜、料酒、加少许盐（提鲜）
  2. 虾倒入锅中煮熟（虾呈弯曲状态（一般5分钟即可））
  3. 出锅冷水冰镇，上盘
可乐鸡翅

大翅、蒜瓣

  1. 焯烫鸡翅：在沸水里，用中火，每次焯烫3~4只鸡翅，30秒后就捞出来

  2. 腌制鸡翅：先在鸡翅上面刮三刀，然后抹盐 生粉 酱油，腌制40分钟，然后倒掉，控干水分

  3. 煎焖鸡翅：倒油下锅，油热之后下鸡翅，中火煎个三分钟，中间要多翻鸡翅避免焦皮。

  4. 加料：加生抽，再下可乐，可乐的量盖到鸡翅即可，可以盖上锅盖焖，偶尔翻翻鸡翅，闻到焦糖味道就是差不多了

面食

荞麦面

荞麦面、青菜、鸡排、鸡蛋、拌面酱

*可搭配一根玉米提供碳水
热量：360kcal；碳水：40g；蛋白质：26g；脂肪：10g；

武汉热干面

碱面、盐、味精、鸡精、胡椒粉、老抽、生抽、辣椒油、香油、花生碎

  1. 煮：锅中加水烧沸，加入少量食用盐，食用油，然后放入面条，边煮边用筷子搅动，等面条煮到7分熟时捞出来，切记不要煮烂了；（折断一节面条看中间有白芯）

  2. 淋：将面条捞出来后过一次凉水，沥干水分后放在案板上，淋上一小勺香油，用筷子拌匀后把面挑散开（这样就不会粘在一起了）

  3. 炒：将辣萝卜还有豆角下到锅中翻炒一下盛起（可以省去这一步，主要是加热后更卫生）

  4. 化：在小碗中倒适量芝麻酱，加温水化开，只要一点点水就可以了，然后加一点盐

  5. 煮：锅里烧水沸腾，把之前淋了香油的面放在锅里经热水烫一下，马上关火，将面捞起来放入碗中（面条已经煮过一次了，所以稍微烫一下就熟啦）

  6. 拌：在碗里加入盐、辣椒油、少许老抽、一小勺生抽、少量醋、鸡精、味精、胡椒粉，再倒入之前化好的芝麻酱，最后搅拌均匀，撒上花生碎、辣萝卜、酸豆角和葱花就可以开吃啦

甜品

绿豆汤

绿豆

  1. 用纯净水煮或是加一点柠檬汁或白醋
    如果有条件，用纯净水煮。如果手边没有纯净水，可以在水里加一点柠檬汁或白醋，以尝不出酸味为宜，这样做只是为了中和酸碱度，出来的汤很绿
  2. 等水沸后再放入绿豆，千万别着急
    不要冷水下锅，而要水煮沸后再把泡发的绿豆下锅
  3. 煮绿豆汤一定要盖上盖儿
    一定要盖上盖子煮，与空气接触越少越好

  4. 多使用压力锅，避免用铁锅
    煮的时间越短、越密封，越能保持绿色，因此用压力锅煮出来的绿豆汤往往保持绿色，避免用铁锅。

  5. 让绿豆快速开花的2个小诀窍
    先浸泡后冷冻法，绿豆速速开花
    把绿豆洗净加少量水（刚好没过绿豆）浸泡十来分钟，然后沥干水分，放入冰箱冷冻层，四小时后拿出，烧一锅沸水，然后放入冻好的绿豆，很快就开花了。

保温瓶里用开水浸泡3~4个小时
将绿豆洗净，放入保温瓶中，倒入开水盖好。3~4个小时后，绿豆粒已涨大变软，再下锅煮，就很容易在较短的时间内将绿豆煮烂

速食

火腿鸡蛋三明治

全麦面包、一个鸡蛋、三个火腿片（无淀粉、低脂）、生菜、西红柿

热量：290kcal；碳水：25g；蛋白质：20g；脂肪：12g；
